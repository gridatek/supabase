## Quick Links

### Common Tasks

| Task | Command |
|------|---------|
| Start development | `npm run dev` |
| Stop services | `npm run stop` |
| Reset database | `npm run reset` |
| Apply migrations | `npm run migrate` |
| Check status | `npm run status` |
| View logs | `npm run logs` |
| Deploy to production | `npm run migrate:prod` |
